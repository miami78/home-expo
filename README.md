
# HomeExpo
A virtual tour of residential homes.

## Setup
Clone the repository.

```bash
git clone https://github.com/fdce/HomeExpo.git

```
`cd` into the directory and install dependencies

```bash
cd HomeExpo
npm install
```
Start
```
npm start
```
